# 💊 Eczane Yönetim ve Reçete Sistemi
Bu proje, Java programlama dilini öğrenirken geliştirdiğim ilk yazılım projemdir. Temel olarak doktor, hasta ve eczacı arasındaki süreci simüle eder.

## 🚀 Öne Çıkan Özellikler
- **Reçete Oluşturma:** Doktor panelinde hasta seçimi yapılarak otomatik reçete oluşturulması.
- **Akıllı Hesaplama:** Sigorta türüne bağlı olarak ilaç fiyatlarında otomatik indirim uygulaması.
- **Esnek Satış:** Reçeteli ilaçların yanı sıra reçetesiz (tezgah üstü) ilaç alım desteği.
- **Sepet Sistemi:** Kullanıcı dostu bir alışveriş sepeti mantığıyla birden fazla ilaç alımı.

## 🛠 Teknik Detaylar
- **Dil:** Java
- **Yöntem:** Nesne Yönelimli Programlama (OOP) prensipleri kullanılarak geliştirilmiştir.
- **Veri Yapıları:** Veriler (şu an için) liste yapıları üzerinde tutulmaktadır. Java Swing / JavaFX
